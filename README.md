# fetchy

**fetchy** is a tiny posix compliant (linux-only) system fetch script. It sacrifices portability for speed, though modifying it to be portable isn't too hard.
![](./preview.webp)

---

## Installation

Add it to any directory in $PATH, typically ~/.local/bin.
```sh
curl -fL https://codeberg.org/oceanicc/fetchy/raw/branch/main/fetchy -o ~/.local/bin/fetchy
chmod +x ~/.local/bin/fetchy
```
