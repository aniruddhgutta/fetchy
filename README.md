# fetchy

**fetchy** is a tiny posix compliant system fetch script. The portable version patches the following functions - `uptime`, `distro` and `kernel`
![](./preview.webp)

---

## Installation

Add it to any directory in $PATH, typically ~/.local/bin.
```sh
curl -fL https://codeberg.org/oceanicc/fetchy/raw/branch/main/fetchy -o ~/.local/bin/fetchy
chmod +x ~/.local/bin/fetchy
```
